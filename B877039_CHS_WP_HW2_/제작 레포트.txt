1. 구현 방법 
_우선적으로 과제 제작에 필요한 툴바, 메뉴바를 수정하는 작업을 진행하였습니다.

진행 방법 : 
1) 리소스 편집기에서 메뉴와 툴바를 추가, ID 설정
2) 클래스 마법사를 통해 ID에 대한 커맨드와 UI UPDATE 함수를 생성

_이후, 본격적으로 기능적인 부분들을 추가하였습니다.

진행 방법:
1) 구현에 필요한 함수들을 클래스 마법사를 통해 생성 ( 키보드 입력, 마우스 클릭 , 타이머 등)
2) 구조체를 이용하여 도형의 세부 정보를 저장 ( 모양, 색상, 위치 등)
3) CList를 이용하여 구조체들을 동적으로 저장
4) 생성한 함수들을 이용하여 도형을 그리고 CList에 저장
5) 지우개의 위치를 이용하여 해당 위치에 그려진 도형들을 탐색하여 삭제
6) 지우개의 위치를 받아, 벽에 부딪히는 경우 이동하는 방향 벡터의 값을 반대로 변환
 
2. 어려웠던 점
1. 잘 다뤄보지 않았던 ID값 부여하는 것과 그와 관련된 함수들을 지정하여 다루는 것이 아직 미숙하여 헷갈리는 점이 있었습니다.
